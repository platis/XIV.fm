# XIV.fm Dalamud repository

Add this URL in Dalamud Settings under **Experimental → Custom Plugin Repositories**:

```text
https://raw.githubusercontent.com/platis/XIV.fm/main/repository/pluginmaster.json
```

This repository currently distributes development prereleases for in-game testing. It is not an official Dalamud plugin repository listing.

Release ZIPs are served as immutable, versioned GitHub release assets. `pluginmaster.json` is generated from the packaged plugin manifest by `scripts/write-pluginmaster.py`.
